# Snow Glide

Snow Glide is a 2D endless runner built with raylib.

## Requirements
- raylib 5.0
- C++14-compatible compiler

## Build
1. Install raylib 5.0 and make sure your compiler toolchain is on `PATH`.
2. From the repository root (`Snow-Glide-main`), run:
   ```
   make
   ```
3. Run the generated binary (`game` or `game.exe` depending on platform).

## Controls
- **Space**: Jump (costs mana)
- **B**: Bankai
- **Mouse**: UI buttons
- **Esc/Backspace**: Return to menu
